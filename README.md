## `lohigh`

DJ Sacabambaspis takes any soundclip *(.wav)* and makes it lo-fi.

![](asset/fish.jpg)

## installation

```console
$ git clone 
$ cd lohigh
$ make config
```

## usage

```console
$ make
$ cd src
$ ./lohigh input.wav output.wav # DJ Sacabambaspis mixes up a lo-fi beat 
$ ./lohigh input1.wav input2.wav output.wav # You are the DJ. 
```
